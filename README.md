- Nginx (Reverse Proxy, Load Balancer & SSL)
- Apache Web Server (Reverse Proxy, Load Balancer & SSL)
- FTP 
- NTP
- MySQL
- DNS
- Jenkins
- Monitoring (Prometheus + Grafana)
- Rsyslog
- Ansible (Linux Server Patching, Manual Patching & App Deployment)
