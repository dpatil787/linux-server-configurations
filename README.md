# Linux Server Configurations

This repository contains practical configurations and setups for Linux-based services.

## Topics Covered
- Nginx (Reverse Proxy, Load Balancer & SSL)
- Apache Web Server (Reverse Proxy, Load Balancer & SSL)
- FTP 
- NTP
- MySQL
- DNS
- Jenkins
- Monitoring (Prometheus + Grafana)
- Rsyslog
- Ansible (Linux Server Patching , Manual Patching & App Deployment) 


Each configuration includes:
- Setup steps
- Configuration files
- Commands used
- Troubleshooting notes
